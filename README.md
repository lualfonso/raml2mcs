# RAML to Spring MCS
Create Java Spring Services from a RAML generated from [yaml2raml](https://github.com/lualfonso/yaml2raml)

## Command

    ruby raml2mcs.rb --from raml_folder --to java_output
    
### Example 
    raml2mcs.rb -f ../yaml2raml/out -t ProjectName
